# SmartCane
Arduino-based smart cane prototype focused on obstacle detection using ultrasonic sensors. Developed as a practical project combining embedded systems, programming, electronics, and accessibility-focused technology.
esigned to assist visually impaired people through real-time obstacle detection.
## Features
- Ultrasonic obstacle detection
- Buzzer alert system
- Real-time distance sensing
- Portable prototype

## Technologies
- Arduino
- HC-SR04 Ultrasonic Sensor
- C++
- Embedded Systems

## Objective
This project was developed as a practical learning experience combining programming, electronics, and accessibility-focused technology.

## Future Improvements
- Vibration feedback system
- Rechargeable battery integration
- Compact and optimized design
- Mobile app connectivity
